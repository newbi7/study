# HTML.Javascript의 MouseEvent
1. mouseenter
들어가는 순간에 발생하는 event
2. mouseover
움직일때마다 발생하는 event
3. mouseleave
나가는 순간 발생하는 event

### Javascript Event
"propagation" 현상이 발생한다.   
이벤트 버블링 - Event Bubbling
이벤트 버블링은 특정 화면 요소에서 이벤트가 발생했을 때 해당 이벤트가 더 상위의 화면 요소들로 전달되어 가는 특성을 의미합니다.
